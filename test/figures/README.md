# Figures for StepSwitch ICLR 2026 Paper

This directory contains all figures for the paper:
**"Step-Level Confidence-Triggered Language Routing for Multilingual Chain-of-Thought Reasoning"**

## Figure Descriptions

### fig1_entropy_analysis.pdf
Per-language step entropy distributions (violin plots). Shows that low-resource languages (Swahili, Bengali) exhibit heavier-tailed distributions, confirming higher uncertainty during CoT reasoning compared to high-resource languages (English, German).

### fig2_main_results.pdf
Main results bar chart: macro accuracy (left panel) and ECE (right panel) for all six methods across ten languages. StepSwitch-1.0 consistently dominates both axes.

### fig3_perlang.pdf
Per-language accuracy grouped bar chart for FixedLang, EnglishOnly, and StepSwitch-1.0. Illustrates that StepSwitch gains are largest for Swahili (+24%), Bengali (+22%), and Thai (+18%).

### fig4_reliability.pdf
Reliability diagrams (calibration plots) for three methods: FixedLang (left), EnglishOnly (center), StepSwitch-1.0 (right). StepSwitch-1.0 achieves the tightest alignment with the perfect calibration diagonal.

### fig5_ablation.pdf
Threshold ablation plots showing accuracy (top row) and ECE (bottom row) for each language group across tau in {0.5, 1.0, 1.5, 2.0}. tau=1.0 is the best macro operating point.

### fig6_switchrate.pdf
Heatmap of per-language, per-threshold switch rates. Swahili and Bengali show 60-92% switch rates even at tau=2.0; English shows near-zero rates across all thresholds.

## Generation

All figures are generated by running `Untitled5.ipynb` in Google Colab.
Figures are saved as PDF (publication quality, vector format).

## Note on Binary Files

The PDF figures are stored in Colab at `/content/figures/` and are committed
to this repository. They can be regenerated at any time by running the notebook.
